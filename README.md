## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

Create a `.env` file, following the model of the `.env.example` file at the project root.

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```
